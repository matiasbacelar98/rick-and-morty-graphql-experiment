# Rick And Morty - Graphql Project

This is a simple project to experiment with graphql and Apollo Client.

## Usage

### Install dependencies

```js
  npm install
```

### Run Server

```js
  npm run dev
```

## Stack

- React
- GraphQL
- Apollo Client
- Tailwind CSS
- Vite

## Features

- Search
- Infinite Scrolling
